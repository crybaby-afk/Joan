# Joan Love Site

A tiny romantic one-page site for Joan.

## Files

- `index.html` - the full site

## Deploy Fast

### GitHub Pages

1. Create a new GitHub repo.
2. Push this folder to that repo.
3. In GitHub, open `Settings` -> `Pages`.
4. Set the source to deploy from the `main` branch and `/root`.
5. GitHub will give you a public link you can send to Joan.

### Netlify

1. Create a new repo on GitHub and push this folder.
2. Log in to Netlify.
3. Choose `Add new site` -> `Import an existing project`.
4. Select the repo.
5. Keep the deploy settings simple:
   - Build command: leave empty
   - Publish directory: `.`
6. Deploy and send her the generated link.

### Vercel

1. Push this folder to GitHub.
2. Import the repo in Vercel.
3. Keep defaults and deploy.

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static file server.
